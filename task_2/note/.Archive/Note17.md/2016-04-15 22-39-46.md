## JSON 转为 Array 数组
- JQ `var arr = $.map(obj, function(el) { return el });`
- 原生写法
 ```
var data = {
    "0": "1",
    "1": "2",
    "2": "3",
    "3": "4"
};
var arr = [];
for (var prop in data) {
    arr.push(data[prop]);
}
console.log(arr);
 ```
